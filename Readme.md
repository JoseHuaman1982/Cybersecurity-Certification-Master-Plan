# Cybersecurity Certification Master Plan (ISC2 CC + CompTIA Security+)

This repository serves as my technical laboratory and study journal, documenting my intensive preparation for the **ISC2 Certified in Cybersecurity (CC)** and **CompTIA Security+ (SY0-701)** certifications.

## 🎯 Project Objective
To bridge the gap between theoretical cybersecurity principles and practical, real-world application. This project demonstrates my ability to configure, audit, and harden network infrastructure using enterprise-grade hardware.

## 📅 12-Week Roadmap (Master Plan)

| Week | Focus Area | Technical Lab (Cisco/Fortinet) |
| :--- | :--- | :--- |
| **1-2** | Foundations, Risk & Governance | Hardening, AAA, VLANs, and BCP/DR |
| **3-4** | Network Infrastructure & Operations | Perimeter Defense, IAM, and Incident Response |
| **5-12** | Security+ Specialization | Wireless, Vulnerability Mgmt, Crypto & Cloud |

## 🛠️ Technical Stack
* **Network Infrastructure:** Cisco 1921 Routers, L2 Switches (Cisco IOS).
* **Perimeter Security:** FortiGate Firewalls (FortiOS).
* **Security Auditing:** Nmap/Zenmap, Vulnerability Scanning.
* **Documentation:** Structured Markdown logs for every lab execution.

## 📂 Repository Structure
* `/docs`: Theoretical notes, domain summaries, and ethics code reviews.
* `/labs`: Detailed execution logs, validation results, and audit reports.
* `/scripts`: Hardening scripts and configuration files (.cfg).
